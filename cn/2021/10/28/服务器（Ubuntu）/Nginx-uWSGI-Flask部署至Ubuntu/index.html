可能遇到的问题

启动uwsgi服务时发现端口被占用

probably another instance of uWSGI is running on the same address (:xxxx).

可以使用如下命令关停该端口：

例如：关停8001端口：sudo fuser -k 8001/tcp

再重新启动uwsgi服务就可以了
